  • OLVÍDATE DEL PIXEL Y PIENSA EN PROPORCIONES
  • Si quieres seguir pensando en pixeles tendras que usar la siguiente formula:

    • Para convertir a ems

      Objetivo en pixeles / Contexto en pixeles = Resultado en Ems

      Ejemplos:
      32px / 16px = 2em
      24px / 16px = 1.5em
      16px / 16px = 1em
      12px / 16px = .75em
      8px / 16px = .5em

    • Para convertira porcentajes

      ( Objetivo en pixeles / Contexto en pixeles ) * 100 = Resultado en Porcentaje

      Ejemplos:
      ( 32px / 16px ) * 100 = 200%
      ( 24px / 16px ) * 100 = 150%
      ( 16px / 16px ) * 100 = 100%
      ( 12px / 16px ) * 100 = 75%
      ( 8px / 16px ) * 100 = 50%

    • Equivalencias

      100% = 1em = 16px = 12pt

  • Uso de porcentajes para el ancho de contenedores
  • Uso de ems o rems para el alto de contenedores, tamaño de fuente, márgenes y rellenos
  • Uso de vh y vw cuando el tamaño dependa del viewport o requieras posicionamiento absoluto basado en el viewport
  • Uso de pixeles para detalles de elementos fijos
  • Contenedores que no se vean afectados por el modelo de caja (margin + padding + border) aplicar box-sizing: border-box
  • Usa la técnica de maquetación que mejor te convenga, aunque te recomendamos usar inline-block para estructuras sencillas y flexbox para estructuras complejas
