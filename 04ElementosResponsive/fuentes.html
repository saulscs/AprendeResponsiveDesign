  • Recuerda siempre asignar un tamaño de fuente base a tu etiqueta html en pixeles 
    					
html { font-size: 16px; }
  • Podemos usar ems que son relativos al tamaño de letra del elemento padre
  • Podemos usar rems que son relativos al tamaño de letra del elemento html
  • ¿Puedo usar rems?
  • Si quieres seguir pensando en pixeles tendras que usar la siguiente formula:

    • Para convertir a ems

      Objetivo en pixeles / Contexto en pixeles = Resultado en Ems

      Ejemplos:
      32px / 16px = 2em
      24px / 16px = 1.5em
      16px / 16px = 1em
      12px / 16px = .75em
      8px / 16px = .5em

    • Equivalencias

      100% = 1em = 16px = 12pt

  • Recuerda que el tamaño de fuente afecta al margin y padding, por lo que es recomendable expresar estas propiedades en ems o rems
  • Recuerda que puedes usar vh y vw cuando el tamaño dependa del viewport o requieras posicionamiento absoluto basado en el viewport
