Cortar la Mostaza

Es una expresión coloquial del inglés que en nuestro idioma significaría:

  • No dar el ancho
  • No estar a la altura
  • No dar la talla

Y en el entorno web se usa para saber si un navegador esta a la altura de las características del estándar actual: 

			
if ( 
	'querySelector' in document && 
	'localStorage' in window &&
	'addEventListener' in window
) {
	CUT THE MUSTARD!				
}
Cortando la Mostaza

¿Porqué esas propiedades?

  • document.querySelector

    La mayoría de librerías JS que manipulan el DOM lo hacen mediante este método. querySelector ha estado disponible desde Firefox 3.5 e IE9.

  • window.localStorage

    Es una de las API's HTML5 más utilizadas, nos permite almacenar información de forma local en el equipo del usuario, con ello las posibilidades son múltiples, desde guardar datos de la sesión actual, configuraciones personales del sitio, hasta contenido dinámico que se pueda mostrar dependiendo del tipo de conexión o dispositivo del que se este navegando.

  • window.addEventListener

    Otro método que usan la mayoría de librerías JS para prender o apagar eventos en el DOM, IE8 no lo soporta.
