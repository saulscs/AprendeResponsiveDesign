Minificación y Concatenación de CSS y JS

regresar

Minificación

Es el proceso para comprimir archivos CSS o JS, que consiste en quitar espacíos en blanco, indentación con la intención de reducir el peso del archivo, por ejemplo la librería jquery en su versión normal pesa alrededor de 240KB y la versión minificada sólo 80KB reduciendo su peso original en una tercera parte.

Concatenación

Es el proceso de unir en un sólo archivo todos los CSS y JS que cargue un sitio con la intención de reducir las peticiones HTTP que se hacen al servidor, por ejemplo si una web tiene 3 hojas de estilo y 4 scripts cuando un usuario acceda a ella hará 7 peticiones HTTP, si concatenamos los archivos en uno sólo, las peticiones se reducirán a 2, una por el CSS y otra por el JS ambos con el contenido unificado de los demás archivos.
