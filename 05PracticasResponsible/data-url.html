Imágenes y vectores como imágenes HTML y fondos CSS con Data URL

Una Data URL es una forma de poner en línea el contenido de una URL a la que se apuntaría normalmente. Todos los navegadores desde IE8 soportan esta técnica.

La ventaja de visualizar nuestras imágenes con Data URL es reducir las peticiones HTTP y eso se verá reflejado en una carga más rápida de nuestro sitio.

Sin embargo, tiene la desventaja de aumentar el tamaño del HTML/CSS considerablemente si se tienen imágenes grandes, por lo que hay que tener cierta precaución al emplear esta técnica, ya que el aumento de tamaño puede sobrepasar a la ganancia en velocidad.

Con esta herramienta podemos convertir imágenes en Data URL. 

			
<img src="data:código-de-la-imagen-o-vector">

<div style="background-image:url(data:código-de-la-imagen-o-vector)"></div>
