Retina Display

Es el concepto que comercialmente define a las pantallas de dispositivos que tienen mayor densidad de pixeles.

El efecto en dichas pantallas hace que todo lo que sea vectorial, como la tipografía, se vea muy nítido y claro; el problema surge con los mapas de bits, mejor conocidos como imágenes, ya que al ser estos definidos por un ancho y alto fijo, en pantallas com mayor densidad tienden a pixelizarse y verse borrosas, por lo que es necesario dar soporte a las imágenes de nuestro sitio para estas pantallas que desde el 2013 existen en computadoras, tabletas y smartphones.

En el sitio de Screen Sizes puedes ver la densidad de pixeles (aspect ratio) de los diferentes dispositivos en el mercado.

Tenemos 4 formas de aplicar imágenes retina:

  1. Con Media Queries CSS

    • Se requiere aplicar una media queries con la propiedad pixel-ratio:1.5
    • ¿Puedo usarla? Hay que aplicarla con sus respectivos prefijos para todos los navegadores
    • Las imágenes hay que definirlas como fondos CSS
    • Cada imágen que se quiera en versión Retina hay que tener una copia físicamente guardada en el mismo directorio que la original y al doble de su tamaño con el mismo nombre seguido del sufijo @2x

    						
.retina
{
	margin: auto;
	height: 256px;
	width: 256px;
	background-repeat: no-repeat;
	background-position: center;
	background-size: 256px 256px;
}

#logo-html5 { background-image: url("img/html5-logo.png"); }

@media 
	only screen and (-ms-min-device-pixel-ratio:1.5),
	only screen and (min--moz-device-pixel-ratio:1.5),
	only screen and (-webkit-min-device-pixel-ratio:1.5),
	only screen and (min-device-pixel-ratio:1.5)
	{
		#logo-html5 { background-image: url("img/html5-logo@2x.png"); }	
}

<div class="retina" id="logo-html5"></div>

  2. Con Retina.JS

    • Hay que cargar en nuestra zona de scripts el archivo de Retina.js y él hace la magía por nosotros
    • Cada imágen que se quiera en versión Retina hay que tener una copia físicamente guardada en el mismo directorio que la original y al doble de su tamaño con el mismo nombre seguido del sufijo @2x
    • En el HTML hay que definir nuestras imágenes con la etiqueta img y cuando Retina.JS detecte que la densidad de pixeles de la pantalla del dispositivo es mayor, reemplazará en el src de nuestra imagen la normal por la @2x.

    						
<img src="img/html5-logo.png">

<script src="js/retina.min.js"></script>

  3. Con CSS puro

    • Sólo hay que tener una versión de la imagen
    • Sólo hay que especificar una regla CSS que defina que todas las imágenes crecerán como máximo a la mitad de su tamaño real
    • Esta técnica te conviene cuando todas las imágenes en tu sitio tienen las mismas dimensiones
    • Si tienes varios tamaños de imágenes tendrás que definir una regla por cada uno
    • El valor de anchura definelo en píxeles para navegadores viejos y en ems o rems para navegadores modernos
    • Ayudate de la formula para convertir pixeles a ems, el resultado divídelo entre 2 por que queremos visualizar la imágen a la mitad de su tamaño

    						
.size256 {
	width: 256px;
	max-width: 16rem;
	height: auto;
}

<img class="size256" src="img/html5-logo@2x.png">
