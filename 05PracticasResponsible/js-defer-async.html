JS Diferido

El atributo defer es booleano, cuando está presente, especifica que el script se ejecuta cuando la página ha terminado de cargarse. Sólo debe usarse para scripts externos.

¿Puedo usarlo? 

				
<script src="js/mi-codigo.js" defer></script>

JS Asíncrono

El atributo async es booleano, cuando está presente, especifica que la secuencia de comandos se ejecutará de forma asíncrona tan pronto como esté disponible. Sólo debe usarse para scripts externos.

¿Puedo usarlo? 

				
<script src="js/mi-codigo.js" aync></script>
